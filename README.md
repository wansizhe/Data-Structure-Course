### 0x00 顺序表
- 基本操作。初始化、第i个元素前插入一个新的元素、删除第i个元素、查找某元素、顺序表的销毁。
- 建立并显示含有n个整数的顺序表。
- 实现对有序表（非递减）插入一个元素的功能。
- 实现两个有序（非递减）表合并生成新的有序表的功能。
- 顺序查找顺序表，删除元素e（只删除找到的第一个值为e的元素）。
- 顺序查找顺序表，删除元素e（删除所有值为e的元素）。
- 实现顺序表的元素逆置。
- 实现顺序表的清空。
- 删除顺序表中所有多余的元素（即没有相同的元素）。
