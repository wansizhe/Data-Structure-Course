## 0x03 约瑟夫问题
设有n个人围坐一圈，从第s个人开始报数，数到第m个人出列，然后从出列的下一个人开始重新报数，数到第m个人又出列。如此反复，直到所有人全部出列。对于任意给定的n,s,m，求按出列次序得到的n个人的序列。
